# Виселица (также «Балда»)

## Принцип игры

Один игрок загадывает слово, а второй игрок пытается его угадать, называя разные буквы. 
Если игрок называет букву, которой в загаданном слове нет, то добавляется одна часть виселицы. 
Игра продолжается до тех пор, пока не разгадают слово или не соберется целиком виселица.

## Запуск программы

Для успешного запуска приложения:

- откройте терминал
- перейдите в папку `hangman`
- скомпилируйте программу:
  - `javac src\*.java -d classes`
- запустите игру `java -Dfile.encoding=UTF-8 -cp classes Main`
